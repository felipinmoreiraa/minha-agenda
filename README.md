# Minha Agenda

Projeto para a disciplina de Projeto Integrado em Programação Web. Uma aplicação web colaborativa para gerenciamento de compromissos e eventos, permitindo cadastro, login, criação/edição/exclusão de eventos, compartilhamento com outros usuários e visualização em calendário interativo com notificações básicas.

## Funcionalidades
- Cadastro e login de usuários com autenticação segura.
- Criação, edição e exclusão de eventos (título, data, hora, local, descrição).
- Compartilhamento de eventos com até 10 usuários.
- Visualização de eventos próprios e compartilhados em um calendário mensal.
- Notificações básicas na interface para eventos próximos.

## Tecnologias
- **Frontend**: HTML, CSS, JavaScript, Vue.js (opcional para o calendário).
- **Backend**: Node.js com Express (APIs REST, autenticação JWT).
- **Banco de Dados**: SQLite (tabelas: Usuários, Eventos, Participantes).
- **Ferramentas**: Git (controle de versão), Trello (gerenciamento), Figma (wireframes), Canva (logo).

## Estrutura do Repositório

- `server/`: Código do backend (Node.js, Express).
- `database/`: Configuração e scripts do SQLite.
